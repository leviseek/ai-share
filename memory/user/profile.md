# 用户画像

## 身份

- 全栈开发者，偏重 TypeScript 全栈、AI Infra 和自动化
- 深度 AI Coding 用户，日常使用 OpenCode + oh-my-openagent 多 agent 编排

## 编码风格

- 先理解根因，再做最小正确改动；不引入无关重构
- 崇尚简单直接，不堆砌抽象层；只在有明确复用价值时才抽取模块
- 优先使用项目已有模式和约定，不引入新风格
- TypeScript 严格模式优先，杜绝 `as any`、`@ts-ignore`

## 工作特点

- 多设备（Windows 主力 + WSL Linux + macOS）
- 多项目同时推进
- 追求长期可维护性，重视配置即代码、可迁移、可审计
- 知识沉淀意识强：结构化记忆 > 聊天记录
